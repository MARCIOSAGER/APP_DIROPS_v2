From src/components/lib/i18n.jsx (11,513 lines — DO NOT read the whole file):

The useI18n hook:
```javascript
export function useI18n() {
  const context = useContext(I18nContext);
  if (!context) throw new Error('useI18n must be used within I18nProvider');
  return context;  // { language, setLanguage, t }
}
// t(key) returns translations[language][key] || key
// If a key is missing it falls back to the key itself — safe to use
```

Import pattern:
```javascript
import { useI18n } from '@/components/lib/i18n';
// Inside component:
const { t } = useI18n();
// Usage:
<h1>{t('page.operacoes.title')}</h1>
```
